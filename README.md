# Customer-Churn-Project
• Built an end-to-end customer churn prediction system for a dataset (3,333 entries, 19 features) with significant class imbalance. 
• Conducted EDA, feature engineering, scaling, and SMOTE to balance the data distribution. 
• Trained multiple models (SVM, KNN, Random Forest, XGBoost) achieving 95% accuracy and 91% F1-score with XGBoost.
